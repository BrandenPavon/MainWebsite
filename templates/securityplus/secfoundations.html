Do this guide in the order that you read it

Use a secure password for your user and for root. A password should have all of these

  • More than 12 characters
  • At least one uppercase
  • At least one lowercase
  • At least one special character
  • Not tied to your Personal Identifiable Information(PII)
  • Not have characters replaced by well known alternatives( 3 instead of E, 0 instead of O)

Lock ssh for root and setup ssh for an unprivileged user

In /etc/ssh/sshd_config, uncomment "PermitRootLogin" if it is commented, and set it to "no".
